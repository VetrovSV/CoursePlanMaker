# 1. Организационно-методический раздел

## 1.1 Цели и задачи дисциплины (модуля)

**Цель изучения дисциплины:**

Формирование у студентов теоретических знаний и практических навыков
проектирования, разработки и эксплуатации реляционных баз данных в
современных СУБД, необходимых для решения профессиональных задач в
области автоматизированных систем и вычислительной техники.

**Задачи изучения дисциплины:**

- изучить основные модели данных, архитектуру и принципы
  функционирования современных СУБД
- освоить язык SQL для создания, модификации и выборки данных
- научиться проектировать реляционные базы данных на основе
  нормализации и ER-моделирования
- получить навыки разработки приложений с использованием баз данных
  и средств доступа к данным

## 1.2 Место дисциплины (модуля) в структуре ОП

Дисциплина относится к обязательной части Блока 1. Изучается на 2-м
курсе в 4-м семестре. Для успешного освоения требуются знания и
навыки, полученные при изучении дисциплин: «Программирование»,
«Объектно-ориентированное программирование», «Информатика».

## 1.3 Объем дисциплины (модуля) с указанием трудоемкости всех видов учебной работы

# 2. Перечень планируемых результатов обучения по дисциплине, соотнесенных с планируемыми результатами освоения образовательной программы

**ОПК-1**

  - ОПК-1.1. Знать: основные принципы построения и функционирования
    современных информационных систем
  - ОПК-1.2. Уметь: проектировать и развивать информационные системы
    для решения профессиональных задач
  - ОПК-1.3. Иметь навыки: разработки и внедрения баз данных в
    различных информационных технологиях
  - **Знать:** основы проектирования и администрирования баз данных,
    принципы организации реляционных и NoSQL СУБД, методы обеспечения
    целостности и безопасности данных
  - **Уметь:** проектировать базы данных, реализовывать их в
    современных СУБД, разрабатывать приложения с использованием баз
    данных
  - **Владеть:** навыками работы с реляционными и распределенными
    базами данных, средствами управления данными в современных ИС

**ОПК-8**

  - ОПК-8.1. Знать: принципы обработки и анализа больших объемов
    данных
  - ОПК-8.2. Уметь: использовать современные СУБД для хранения и
    обработки данных
  - ОПК-8.3. Иметь навыки: проектирования и эксплуатации баз данных
    в промышленных приложениях
  - **Знать:** модели данных, язык SQL, методы индексирования и
    оптимизации запросов, принципы резервного копирования и
    восстановления данных
  - **Уметь:** разрабатывать SQL-запросы, проектировать схемы баз
    данных, обеспечивать их производительность
  - **Владеть:** навыками анализа и обработки данных в различных
    типах СУБД

**ПК-1**

  - ПК-1.1. Знать: принципы проектирования и разработки
    автоматизированных систем управления
  - ПК-1.2. Уметь: разрабатывать информационное обеспечение
    автоматизированных систем
  - ПК-1.3. Иметь навыки: интеграции баз данных с прикладными
    модулями АСУТП
  - **Знать:** особенности проектирования баз данных для
    промышленных систем, методы интеграции с SCADA-системами
  - **Уметь:** разрабатывать базы данных для автоматизированных
    систем управления технологическими процессами
  - **Владеть:** навыками разработки информационного обеспечения для
    промышленных автоматизированных систем

# 8. Методические рекомендации по организации изучения дисциплины

Лекционные занятия. Рекомендуется вести краткий конспект со слайдов
во время занятия. Перед занятием рекомендуется повторить пройденный
материал, с использованием конспектов, слайдов лекций, рекомендованных
и самостоятельно найденных источников; познакомится с планом
предстоящего занятия. Подготовить вопросы по пройденным темам.

Лабораторные занятия. Цель занятий — углубление и закрепление
теоретических знаний, формирование у студентов определенных умений и
навыков. Перед занятием студент должен выполнить задания.
Подготовится к защите работы, знать ответы на вопросы к заданию.
Рекомендуется доработать конспект. Желательно использовать несколько
источников.

Самостоятельная работа. Приводит студента к получению нового знания,
упорядочению и углублению имеющихся знаний, формированию у него
профессиональных навыков и умений.

Рекомендуется с самого начала освоения курса работать с
рекомендованными источниками и предлагаемыми заданиями в форме
подготовки к очередному аудиторному занятию, дорабатывать конспект во
время самостоятельной работы.

По заданиям, предлагаемым для решения на лабораторных занятиях,
студент должен отчитаться до наступления сессии.
